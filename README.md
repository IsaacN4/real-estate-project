## Real Estate Website

<p>
  This project is a responsive real estate website built using HTML and CSS, designed to showcase property listings and agent profiles. 
  The website features a clean, user-friendly interface, including sections for property details, agent contact information, and property 
  search functionality. It is optimized for different screen sizes to ensure a seamless experience on both desktop and mobile devices.
</p>

![Picture of the main page](https://github.com/IsaacN4/real-estate-project/blob/main/assets/img/image1.png?raw=true)

## Technologies Used

- **HTML5**: Structured the content and layout of the web pages.
- **CSS3**: Styled the website, including layout, colors, and typography.
- **Responsive Web Design**: Ensured compatibility across different screen sizes using media queries.
- **Flexbox/Grid**: Implemented flexible layouts for property listings and agent profiles.
- **Semantic HTML**: Used meaningful tags to improve accessibility and SEO.
